### Reflection Questions
---
***Consider these questions as you work through the lab:***

***Branch Design Challenge: What was the most challenging part of designing the conversation branches? Did you discover anything about how voice agents handle different conversation paths?***
+ The most challenging part of the assignment was getting the correct triggers to forward the conversation to the correct sub agent. I picked a very niche topic, which I think hindered me specifically with the word "language". As it created a broad topic of other languages, instead of the user asking bout either English or Na'vi.
+ I discovered that outlining how they should treat the conversation with the user greatly effects how they handle the conversation paths. I noticed when you changed the conversation test length to a longer exchange you can really see how the voice agent applies it when you outline for example empathetic or angry. 

***Knowledge Base Impact: How did the knowledge base content you added affect the agent's responses? Did certain types of content work better than others?***
+ My knowledge base affected the agents response in a positive way, as it enabled the AI to reference and expand on the topics or words asked about.
+  I think the two language books I included definitely helped the "Language Teachers" effectively tutor and inform the user. The fan made history articles on the history of the franchise though could have been better as a lot of the examples had spelling or incorrect information referenced. 

***Real-World Application: What real-world application could this voice agent have? Think about a specific industry, business, or use case where this type of agent would be valuable***
+ Personally I think this type of agent would be a great set up to teach language and history topics or in my case help a fan acquire and condense knowledge about their favorite franchise. I think it can be very interchangeable and profitable with the right application.
